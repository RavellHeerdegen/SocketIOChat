# SocketIOChat
Some simple chat with some cool features

# Requirements
 - NodeJS

# Installation
## Frontend
Run following Command in the Frontend-Folder to install all dependencies
```bash
npm install
```

# Running the Project

## Frontend
Run following Command in the Frontend-Folder to start the server
```bash
npm start
```
